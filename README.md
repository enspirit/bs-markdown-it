# bs-markdown-it

[BuckleScript](https://bucklescript.github.io) bindings for [markdown-it](https://github.com/markdown-it/markdown-it).

# Installation

```
npm i bs-markdown-it -s
```

# Usage

## Basic usage

```reasonml
MarkdownIt.render("# This is a markdown title")
```

# Available Options

- [ ] render: option(string)
